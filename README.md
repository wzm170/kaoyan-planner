# 国科大上海有机所考研指挥中心

## 本机打开

直接双击 `index.html`。

## 同一 Wi-Fi 访问

运行：

```powershell
.\start-lan.ps1
```

然后在其他设备访问电脑的局域网地址，例如：

```text
http://192.168.1.4:5173
```

## 不同网络访问

### 临时公网地址

运行：

```powershell
.\start-public.ps1
```

终端会输出一个 `https://...loca.lt` 公网地址。电脑保持开机、窗口保持运行时，手机流量、异地电脑都可以打开。

### 固定网址

推荐发布到 GitHub Pages。发布后网址通常是：

```text
https://你的GitHub用户名.github.io/kaoyan-planner/
```

固定网址不依赖本机开机，适合长期使用。

## 学习记录

学习记录默认保存在当前浏览器。跨设备迁移时，在“学习记录”页使用“导出记录”和“导入记录”。

如果需要真正的多设备自动同步，需要再接入数据库或云表格。
