# 考研长程备考指挥中心

## 本机打开

直接双击 `index.html`。

## 同一 Wi-Fi 访问

运行：

```powershell
.\start-lan.ps1
```

然后在其他设备访问电脑的局域网地址，例如：

```text
http://192.168.1.4:5173
```

## 不同网络访问

### 临时公网地址

运行：

```powershell
.\start-public.ps1
```

终端会输出一个 `https://...loca.lt` 公网地址。电脑保持开机、窗口保持运行时，手机流量、异地电脑都可以打开。

### 固定网址

推荐发布到 GitHub Pages。发布后网址通常是：

```text
https://你的GitHub用户名.github.io/kaoyan-planner/
```

固定网址不依赖本机开机，适合长期使用。

## 学习记录

学习记录默认保存在当前浏览器。跨设备迁移时，在“记录”页使用“导出记录”和“导入记录”。

## 当前功能

- 首次进入先填写目标院校、专业和方向。
- 内置国科大上海有机所 2026 招生信息：101 政治、201 英语一、619 物理化学(甲)、820 有机化学。
- 每日自动换一句学习激励语。
- 每日学习记录支持上传图片。
- 内置免费规则 AI 分析学习记录。
- 每周四科真题演练。
- 考试总结区域。
- GitHub Actions 每日自动检索招生信息，有变化才提醒。

## 云同步与账号

GitHub Pages 不能自己保存账号数据。要实现“同一个账号下跨设备同步记录和图片”，推荐使用 Supabase 免费项目：

1. 创建 Supabase 项目。
2. 执行 `supabase-schema.sql`。
3. 创建 Storage bucket，例如 `record-images`。
4. 在前端接入 Supabase URL 和 anon key。

当前版本已经保留“账号与同步”界面，但默认仍是本机模式。

如果需要真正的多设备自动同步，需要再接入数据库或云表格。
